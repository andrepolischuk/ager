# Ager

  Age calculations

## Instalation

  Browser:

```html
<script src="https://cdn.rawgit.com/andrepolischuk/ager/1.2.0/ager.min.js"></script>
```

  Component(1):

```sh
$ component install andrepolischuk/ager
```

  Npm:

```sh
$ npm install ager
```

## API

### ager(date)

  Calculate age via date

```js
ager(new Date());
```

### ager(year, month, day)

  Calculate age via arguments

```js
ager(1991, 7, 24)
```

### ager(array)

  Calculate age via array

```js
ager([1991, 7, 24])
```
